# Fun2Link by ZIO Interactive
Ported to java.awt from Samsung X120
 
Uses [https://github.com/umjammer/vavi-sound/] to play sound

# Release

https://github.com/steelswing/fun2link-java/releases/download/1.0/Fun2Link.jar

 
# Input

W, A, S, D or Arrows - Move cursor

Enter, Space - Select and place tube 

Q - Fast
E - Main menu
